02

One queue. One brain.

Most MSPs run dispatch out of three browser tabs and a chat thread. Nova replaces it with a single ranked queue, a live status board, and an attention list that flags trouble before it bites.

06

The system gets sharper every week.

Every dispatcher correction is a signal. Nova captures them all and surfaces auto-proposed rules for review — no model retraining, no data scientist, no waiting for a quarterly release.
